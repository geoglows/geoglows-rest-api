Welcome to Version 2 of the GEOGLOWS Streamflow Service! Here is what's new:

GEOGLOWS Stream Network
  • New streams based on TDX Hydro.
  • ~7 million stream reaches.
  • 125 Vector Processing Units (VPUs).
  • Better estimates for routing parameters and travel times.
Longer Retrospective Simulation
  • Eighty (80) years of retrospective data.
  • Parameters to specify start and end dates.
  • Computed and updated with less than a month lag.
Modeling Workflow Improvements
  • Faster simulations.
  • Finer temporal and spatial resolutions.
  • Changes to file formats used for improved retrieval speed (NetCDF to Zarr).
REST API Service Improvements
  • REST API versioning.
  • Updates to read result's new Zarr format.
  • New REST API endpoints to facilitate data retrieval.

Version 1 results will continue to be available until April, 2025. For more information join our Google Group and Mailing List Here.
