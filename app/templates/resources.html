Resources

Several training resources are available to help you learn to use the GEOGLOWS ECMWF Streamflow model.

Python client - geoglows

There is an official python client for the GEOGLOWS data service. This python package contains functions for retrieving data from the REST service, generating plots, and performing some post processing.

  • Documentation, tutorials, example code: training.geoglows.org.
  • Source code on GitHub: https://github.com/geoglows
  • Conda-forge page: https://github.com/conda-forge/geoglows-feedstock
  • Pypi distribution: https://pypi.org/project/geoglows/

Data Service

  • Source Code for the Data Service: github.com/geoglows/geoglows-rest-api

GEOGLOWS Projects

GEOGLOWS has been working on several other tools to help people access water data, including the GEOGLOWS ECMWF Streamflow Model. You can find presentation slides, demo videos, workshops, tutorials and other documentation content for this streamflow model and other GEOGLOWS Projects at training.geoglows.org.

More information about GEOGLOWS can also be found at geoglows.org.
